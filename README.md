## Getting Started

First, install all dependencies:

```bash
npm install
```

Next, create a config file:

```bash
cp .env.example .env
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
